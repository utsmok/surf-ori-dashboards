# ORI Dashboards on research in the Netherlands

This repository hosts a number of [Marimo-based](https://marimo.io) dashboards that can be viewed on [GitHub Pages](https://surf-ori.github.io/dashboards/). The Marimo notebooks are in the `notebooks` directory.

The notebooks are turned into WASM+HTML based dashboards using GitHub Actions following the [marimo WebAssembly + GitHub Pages Template](https://github.com/marimo-team/marimo-gh-pages-template). All the files relevant for that can be found under the `.github` directory.
